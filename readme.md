Delani Studios

This project was created by Alice Githui.

## Description
This is a website for Delani Studios, an innovative and dynamic technology company that offers tailored software solutions to their clients to meet their particular needs. The website showcases the previous works done, the companies that they have worked with, contact details and the socials where the company is available.

## Technologies Used

-HTML
-CSS
-JavaScript/jQuery

## Installation and Set-up

To acquire a local copy:

- Clone the repository
- Navigate to the parent folder
- Open the index.html file
